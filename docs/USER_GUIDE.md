## 用户注册

### 注册方式

支持手机号或邮箱注册。

### 注册步骤

1. 打开 KeBaiPay 应用
2. 点击"注册"
3. 填写以下信息：
   - 昵称（必填）
   - 手机号 或 邮箱（至少提供一个）
   - 密码（至少 8 位，包含大写字母、小写字母、数字中的至少两类）
4. 点击"注册"

### 密码要求

- 至少 8 位
- 包含大写字母（A-Z）
- 包含小写字母（a-z）
- 包含数字（0-9）
- 至少满足以上两类

### 注册示例

```json
{
  "nickname": "张三",
  "phone": "13800138000",
  "password": "MyPassword123"
}
```

## 身份认证

### 为什么需要实名认证？

- 解锁转账功能
- 解锁提现功能
- 提高账户安全等级
- 满足监管要求

### 认证步骤

1. 进入"我的" → "实名认证"
2. 填写真实姓名
3. 填写身份证号码
4. 提交认证

### 认证状态

| 状态 | 说明 |
|------|------|
| UNVERIFIED | 未认证 |
| PENDING | 审核中 |
| VERIFIED | 已认证 |
| REJECTED | 认证被拒绝 |

### 认证示例

```json
{
  "realName": "张三",
  "idCard": "110101199001011234"
}
```

## 账户充值

### 充值方式

- 支付宝充值
- 微信支付

### 充值步骤

1. 进入"我的钱包"
2. 点击"充值"
3. 输入充值金额
4. 输入支付密码
5. 选择支付方式
6. 完成支付

### 充值限制

- 单笔最小充值金额：0.01 元
- 单日充值限额：根据账户等级不同

### 充值示例

```json
{
  "amount": 100.00,
  "payPassword": "pay_password",
  "idempotencyKey": "recharge_001"
}
```

## 转账

### 转账方式

- 通过用户 ID 转账
- 通过扫码转账

### 转账步骤

1. 进入"转账"
2. 输入收款用户 ID 或扫描收款码
3. 输入转账金额
4. 输入支付密码
5. 确认转账

### 转账限制

- 单笔最小转账金额：0.01 元
- 单日转账限额：根据账户等级不同
- 收款方必须完成实名认证

### 转账示例

```json
{
  "toUserId": "target_user_uuid",
  "amount": 50.00,
  "payPassword": "pay_password",
  "remark": "转账备注"
}
```

### 错误处理

| 错误码 | 说明 | 解决方法 |
|--------|------|----------|
| KB501 | 转账金额无效 | 检查金额是否大于 0 |
| KB502 | 不能给自己转账 | 选择其他收款方 |
| KB005 | 余额不足 | 先充值再转账 |
| KB214 | 对方未实名 | 提醒对方完成实名认证 |

## 提现

### 提现方式

- 银行卡提现

### 提现步骤

1. 进入"我的钱包"
2. 点击"提现"
3. 输入提现金额
4. 输入支付密码
5. 选择提现渠道
6. 确认提现

### 提现限制

- 单笔最小提现金额：0.01 元
- 单日提现限额：根据账户等级不同
- 必须完成实名认证

### 提现示例

```json
{
  "amount": 50.00,
  "payPassword": "pay_password",
  "channel": "BANK",
  "channelAccount": "6222021234567890123"
}
```

### 提现状态

| 状态 | 说明 |
|------|------|
| PENDING | 等待审核 |
| PROCESSING | 处理中 |
| SUCCESS | 提现成功 |
| FAILED | 提现失败 |
| REJECTED | 提现被拒绝 |

## 红包

### 发红包

#### 步骤

1. 进入"红包"
2. 点击"发红包"
3. 输入红包总金额
4. 输入红包个数
5. 填写祝福语（可选）
6. 确认发送

#### 示例

```json
{
  "amount": 10.00,
  "count": 5,
  "remark": "恭喜发财",
  "expiresAt": "2024-01-02T00:00:00.000Z"
}
```

### 领红包

#### 步骤

1. 扫描红包二维码或点击红包链接
2. 点击"领取"
3. 红包金额自动存入钱包

#### 限制

- 不能领取自己发的红包
- 红包过期后无法领取
- 每个红包每人只能领取一次

### 红包状态

| 状态 | 说明 |
|------|------|
| PENDING | 待领取 |
| RECEIVED | 已领取 |
| EXPIRED | 已过期 |

## 收款码

### 个人收款码

#### 获取收款码

1. 进入"收款码"
2. 系统自动生成个人收款码
3. 分享给付款方

#### 收款码信息

```json
{
  "id": "uuid",
  "code": "QR_xxxxx",
  "type": "PERSONAL",
  "status": "ACTIVE"
}
```

### 固定金额收款码

#### 创建步骤

1. 进入"收款码"
2. 点击"创建固定金额收款码"
3. 输入金额
4. 填写备注（可选）
5. 确认创建

#### 示例

```json
{
  "amount": 10.00,
  "remark": "午餐费"
}
```

### 扫码付款

#### 步骤

1. 扫描对方收款码
2. 确认收款信息
3. 输入支付密码
4. 完成付款

#### 限制

- 不能扫描自己的收款码
- 收款码有效期内才能使用

## 账单管理

### 查看账单

1. 进入"账单"
2. 查看所有交易记录

### 账单筛选

- 按类型筛选：收入 / 支出
- 按时间筛选：今日 / 本周 / 本月 / 自定义

### 账单类型

| 类型 | 说明 |
|------|------|
| RECHARGE | 充值 |
| WITHDRAW | 提现 |
| TRANSFER | 转账 |
| RECEIPT | 收款 |
| PAYMENT | 支付 |
| REFUND | 退款 |
| RED_PACKET | 红包 |

### 账单示例

```json
{
  "id": "uuid",
  "type": "TRANSFER",
  "direction": "INCOME",
  "amountYuan": "50.00",
  "counterparty": "张三",
  "remark": "转账",
  "createdAt": "2024-01-01T00:00:00.000Z"
}
```

## 常见问题

### Q: 如何重置支付密码？

1. 进入"我的" → "设置"
2. 点击"重置支付密码"
3. 输入旧支付密码
4. 输入新支付密码
5. 确认重置

### Q: 支付密码锁定后如何解锁？

支付密码连续错误 5 次将锁定 15 分钟。等待 15 分钟后自动解锁。

### Q: 如何查看当日限额？

进入"我的" → "限额查询"，查看当日已用/总额度。

### Q: 转账需要手续费吗？

目前转账免手续费。

### Q: 提现多久到账？

- 审核通过后，通常 1-3 个工作日到账
- 具体到账时间取决于银行处理速度

### Q: 如何联系客服？

- 在线客服：应用内"我的" → "在线客服"
- 邮箱：support@kebaipay.com
